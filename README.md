# VPN Telegram Bot

Бот для продажи VPN-доступа через Telegram. Пользователи выбирают сервер и срок подписки, оплачивают через интегрированную платёжную систему и получают VPN-ключ.

## Возможности

- Выбор сервера: Европа, США, Азия.
- Срок подписки: 1 неделя, 1 месяц, 3 месяца.
- Автоматическая генерация и отправка ключа после оплаты.
- Админ-панель: статистика пользователей, продаж, активных ключей.
- База данных in-memory (для демонстрации; в production заменить на постоянную).

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/sergeyzhinskiy/sale_vpn.git
   cd sale_vpn```

2. Установите зависимости:

```bash
pip install telethon
```

3. Создайте файл config.ini (пример):

```ini
[Telegram]
api_id = YOUR_API_ID
api_hash = YOUR_API_HASH
bot_token = YOUR_BOT_TOKEN
admin_ids = YOUR_TELEGRAM_ID
```

4. Запустите бота:

```bash
python vpn_bot.py
```

# Настройка перед запуском

**_Telegram API_**

Перейдите на my.telegram.org, войдите, создайте приложение.

Скопируйте api_id и api_hash.

Создайте бота через @BotFather и получите bot_token.

Узнайте свой Telegram ID (например, через @userinfobot) и укажите в admin_ids.

**_Платёжная система (для реальной работы)_**

В текущей версии используется заглушка. Для приёма реальных платежей замените функцию проверки check_payment_ на вызов API вашего провайдера (YooKassa, Stripe, CryptoBot). Пример для YooKassa:

```python
# В обработчике check_payment_
payment_status = yookassa_client.get_payment(payment_id).status
if payment_status == 'succeeded':
    # активировать ключ
```

VPN-сервер (реальная генерация ключей)

Сейчас ключи генерируются как случайные строки и никуда не передаются. Для реальной работы:

**Настройте сервер с WireGuard/OpenVPN.**

Напишите скрипт, который создаёт нового пользователя и возвращает конфигурационный файл.

Замените функцию generate_vpn_key на вызов этого скрипта.

# Использование

**Пользователь**

Отправьте боту /start.

Нажмите «**Купить VPN**» → выберите сервер → выберите срок.

Перейдите по ссылке оплаты (в демо-версии это example.com).

После реальной оплаты нажмите «Я оплатил».

Получите VPN-ключ и инструкцию в личные сообщения.

**Администратор**

В главном меню появится кнопка «Админ панель».

Доступна статистика: общее число пользователей, активных ключей, продаж, доход.

**Структура проекта**

```text
vpn_bot.py          # основной код бота
config.ini          # конфигурация (не в репозитории)
```
